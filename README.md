## 📌 Descrição do Projeto
Este é o meu portfólio pessoal construído do absoluto zero. O objetivo principal deste projeto foi consolidar os fundamentos do Front-End (HTML, CSS e JavaScript) sem a dependência de frameworks, bibliotecas ou geradores de código, priorizando a semântica, performance e acessibilidade.

## 🛠️ Tecnologias e Técnicas Utilizadas
Este projeto seguiu restrições técnicas rigorosas, utilizando apenas tecnologias nativas da web:

- **HTML5:** Estrutura altamente semântica (`<header>`, `<nav>`, `<main>`, `<section>`).
- **CSS3 Moderno:** 
  - Layout construído com **Flexbox** e **CSS Grid**.
  - Gerenciamento de design system utilizando **CSS Variables** (Custom Properties).
  - Animações e microinterações criadas via **CSS Hand-coded** (`@keyframes` e `transitions`), sem uso de bibliotecas de terceiros.
- **JavaScript (Vanilla):** Scripts puros para manipulação de DOM e interatividade do layout.
- **Design:** Abordagem **Mobile-First** com Media Queries para responsividade total de 320px a 1920px.

> **Nota:** Zero uso de frameworks como Tailwind, Bootstrap, React ou jQuery.
